My Projects

Low-Frequency Trading Algorithm

Using historical data of six companies, I trained LSTM models to predict stock prices. I then implemented a simple algorithmic trading strategy that makes buy or sell decisions based on predicted prices.

Easy Accounting

The EasyAccounting application is a tool designed to automate the processing of financial data for MetroTek, an electrical services company I worked with. Built with Tkinter for the graphical user interface and leveraging libraries like Pandas, OpenPyXL, and Glob, this app effectively streamlined the accounting department's workflow.

Optimizing P300 Classifiers

Exploring intended vs. non-intended EEG responses: A comparison of a logistic regression, linear discriminant analysis and support vector machine based signal classifiers to improve the efficiency of a P300 brain-computer interface.
